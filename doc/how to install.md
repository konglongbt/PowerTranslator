# 安装

1. 关闭 PowerToys
2. 下载 [Translator.zip](https://github.com/N0I0C0K/PowerTranslator/releases)
3. 解压到`%LOCALAPPDATA%\Microsoft\PowerToys\PowerToys Run\Plugins`.
   如图所示
   ![file](../Images/file.png)

4. 启动 PowerToys

## 如何升级

1. 关闭 PowerToys
2. 下载新版本`Translator.zip`.
3. 解压覆盖掉原先的 Translator 文件夹即可.
4. 启动 PowerToys
